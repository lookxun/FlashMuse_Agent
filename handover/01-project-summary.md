# Project Summary

## 项目目标

做一个内部使用的“简单版即梦”，核心体验是：

1. 直接进入主页，不登录
2. 用户在聊天框里输入需求
3. 系统理解上下文
4. 系统自动优化提示词
5. 可以生成图片或视频
6. 保存历史记录

当前阶段：

- 先把核心流程跑通
- 先给内部人员测试
- 后面再补登录、支付、正式推广能力

## 当前技术栈

- 框架：`Next.js 16`
- 前端：`React 19`
- 样式：`Tailwind CSS 4`
- 包管理：`npm`
